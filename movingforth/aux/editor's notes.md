# Editor's Notes

- The documents were edited using the [Mark Text](http://marktext.app/) editor
- Tables are created or edited in Libreoffice Calc (LC) using an html template. Once ready, just: (i) save in CSV format using the {space} as delimiter, without string delimiter, and leaving the other options unchecked; or (ii) direct ctrl-c/ctrl-v from LC to the text file (markdown)
- Figures are edited using Libreoffice Drawn and exported as SVG. In trial-and-error tests, what looks best on both the laptop monitor and the smartphone screen are ~7 x 3-inch-sized pictures with font size 10
- The original documents were converted from html to markdown using pandoc
