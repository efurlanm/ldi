# Editor's Notes

- The documents were edited using the [Mark Text](http://marktext.app/) editor
- Tables are created or edited in Libreoffice Calc using an html template. Once ready, just save in CSV format using the {space} as delimiter, without string delimiter, and leaving the other options unchecked
- Figures are edited using Libreoffice Drawn and exported as SVG. In trial-and-error tests, what looks best on both the laptop monitor and the smartphone screen are ~18 x 7-inch-sized pictures with font size 10
