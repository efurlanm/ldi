# Editor's Notes

- The documents were edited using the [Mark Text](http://marktext.app/) editor
- Tables are edited in Libreoffice Calc (LC) using an html template. Once ready, just ctrl-c/ctrl-v from LC to the text
- Figures are edited using Libreoffice Drawn (LD) and exported as SVG. In trial-and-error tests, what looks best on both the laptop monitor and the smartphone screen are ~7 x 3-inch-sized pictures with font size 10
- The original documents were converted from html to markdown using `pandoc`
- PDF files were converted to TXT using `pdftotext` (poppler-utils)
